# M5 — AI 활용 전략 수립 (11–12주) [구조 스텁]

> 코스: SAI-501 | 상태: 콘텐츠 작성 예정 (course_package v0.1.0 기준 구조만 확정)

## 핵심 질문

업무별 AI 적용 가능성을 어떻게 평가하고 전략을 수립하는가? 어떤 업무는 자동화하고, 어떤 업무는 보조에 그치고, 어떤 업무는 사람 검토 경계 안에 두는가?

## 주차

- 11주: 시뮬레이터 실습 — 행정업무 프로세스 시뮬레이터(D-2 스키마, 6차원 루브릭)로 업무 1건의 AI 지원 시나리오 구성
- 12주: 전략 수립 — AI 전략 검증 패킷(AIC-0001~0030, 허용 자동화 수준 5단계)을 기준으로 자기 업무 인벤토리의 AI 적용 가능성 평가
