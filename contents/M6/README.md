# M6 — 혁신 계획 수립 (13–15주) [구조 스텁]

> 코스: SAI-501 | 상태: 콘텐츠 작성 예정 (course_package v0.1.0 기준 구조만 확정)

## 핵심 질문

M1–M5의 분석(실태→원인→담당 구조→파이프라인→AI 전략)을 통합하여, 소속 학교 맥락에 적용 가능한 "인공지능 활용 행정업무 혁신 계획서"를 어떻게 완성하는가?

## 주차

- 13주: 계획서 작성 — 백서 9단계 카드 구조를 참조한 계획서 초안 작성 (비식별 원칙 준수)
- 14주: 동료평가 — 6차원 루브릭 기반 상호 검토, 사람 검토 경계 설정의 타당성 점검
- 15주: 최종 발표 — 혁신 계획서 발표와 종합 피드백
